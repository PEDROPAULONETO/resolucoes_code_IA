# Resoluções de Códigos em Python com Inteligência Artificial

Este projeto demonstra a resolução de problemas básicos de programação em Python, com o auxílio de uma Inteligência Artificial para gerar e otimizar o código. Cada arquivo `.py` representa um desafio diferente.

## O que foi aprendido?

Através da criação dos scripts neste projeto, foram praticados os seguintes conceitos fundamentais de Python:

1.  **Entrada e Saída de Dados:**
    *   Uso da função `input()` para receber dados do usuário.
    *   Uso da função `print()` para exibir resultados, incluindo o uso de f-strings para formatação.

2.  **Tipos de Dados e Conversões:**
    *   Manipulação de strings (`str`).
    *   Uso de números inteiros (`int`) e de ponto flutuante (`float`).
    *   Conversão de tipos (type casting) com `int()` e `float()` para processar entradas numéricas.

3.  **Operadores Aritméticos e de String:**
    *   Soma (`+`), Subtração (`-`), Multiplicação (`*`), Divisão (`/`) e Exponenciação (`**`).
    *   Concatenação de strings (`+`).
    *   Repetição de strings (`*`).
    *   Operador de módulo (`%`) para verificar se um número é par ou ímpar.

4.  **Estruturas Condicionais:**
    *   Uso de `if`, `elif` e `else` para controlar o fluxo do programa com base em decisões.
    *   Implementação de uma expressão condicional (operador ternário) para uma verificação concisa (`'par' if numero % 2 == 0 else 'ímpar'`).

5.  **Tratamento de Exceções:**
    *   Uso do bloco `try...except` para lidar com erros em tempo de execução (como `ValueError` quando o usuário digita um texto em vez de um número), tornando os programas mais robustos.

## Arquivos do Projeto

*   `concat_dados.py`: Solicita dois dados do usuário e os concatena em uma única string.
*   `ope_mat.py`: Solicita dois números e permite ao usuário escolher uma operação aritmética (soma, subtração, multiplicação ou divisão) para realizar entre eles.
*   `par_impar.py`: Recebe um número inteiro e verifica se ele é par ou ímpar.
*   `media_notas.py`: Calcula a média aritmética de três notas fornecidas pelo usuário.
*   `repet_txt.py`: Solicita uma base e um expoente e calcula a potência.
